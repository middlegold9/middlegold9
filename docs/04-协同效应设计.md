# 04 · 协同效应设计（项目灵魂）

> 这是整个 FDE 项目最核心的价值：把 7 个「有交集但各自为政」的板块，用规则引擎串成一张协同网。

## 协同地图

```
            ┌───────────────── 土拍市场 ─────────────────┐
            │ 楼面价/溢价率                                │
            ▼                                            ▼
      自动生成选题                                  区域调价提示
      (自媒体内容)                                  (在售房源)
            │                                            │
            ▼                                            ▼
        引流线索 ──► 客户池 ◄────── 二手成交（大额）────► 客户价值评分
            │            │                                │
            │            ▼                                ▼
            │      客户360画像                       自动升级 VIP
            │            │                                │
            │            ▼                                ▼
            └─────► 会面 Brief ◄──────────────── VIP 资产配置漏斗
                    (拉全板块)                  (理财/保险/身份/教育)
                                                          │
                                                          ▼
                                                    合同自动生成
```

## 四类协同规则详解

### 规则 1：成交 → 客户价值评分 → 自动升级 VIP 漏斗
**触发**：任何成交记录写入。
**逻辑**（`score_client`）：
```
价值分 = 成交额贡献(封顶40) + 净资产(封顶25) + 需求广度(每项+6) + 多区域兴趣(封顶9)
≥70 → VIP   ≥40 → 高净值   其余 → 普通
```
**协同动作**：升级到 VIP 时，自动补全资产配置需求待办，并写协同日志。

> 业务价值：买了豪宅的大客户**不会再被遗漏**，自动进入「理财/保险/香港身份/子女教育」的高价值服务漏斗。

### 规则 2：土拍/豪宅 → 自动生成自媒体选题
**触发**：土拍记录 / 豪宅房源写入。
**逻辑**（`generate_topics_from_auction`）：
- 每条土拍 → 1 条「土拍分析」选题
- 溢价率 ≥15% → 追加 1 条「区域解读」爆款选题
- 豪宅房源 → 1 条「实探看房」选题

> 业务价值：选题从「等灵感」变成「数据一来就有」，且自带数据论据，更易出爆款。

### 规则 3：区域情报聚合 → 调价提示
**触发**：访问区域情报看板。
**逻辑**（`district_intelligence`）：聚合福田/南山/宝中的成交均价、在售挂牌价、最新土拍楼面价。
**关键判断**：
```
若 最新土拍楼面价 > 在售均价：
   → "面粉贵过面包，在售二手房源建议挺价/上调报价"
```

> 业务价值：把土拍数据（拍视频用的）反哺到二手成交（赚佣金的），实现「内容板块」与「交易板块」的协同。

### 规则 4：会面 → 自动生成 Brief
**触发**：创建 VIP 会面。
**逻辑**（`build_meeting_brief`）：拉取该客户的成交历史、已配置/未配置需求、跨板块推荐，生成一页纸 brief。

> 业务价值：会面前 0 准备成本，且能精准识别「本次该切入哪个机会点」。

## 跨板块推荐（recommend_cross_module）

给定任一客户，引擎推荐他可能需要的其它板块服务：

| 条件 | 推荐 |
|------|------|
| 累计成交 ≥1500万 且 未配理财 | → 理财产品配置 |
| 累计成交 ≥2000万 且 未配身份 | → 香港身份规划 |
| 未配保险 | → 大额保单 |
| 净资产 ≥3000万 且 未配教育 | → 子女教育规划 |
| VIP 且 无豪宅标签 | → 邀约豪宅看房置换 |

## 协同的可审计性

每一次自动联动都写入 `SynergyEvent`，在前端「协同事件流」实时可见：
```
二手成交/资产 → VIP资产配置 [客户升级] 客户「陈总」价值分92分，由普通升级为VIP…
土拍市场 → 自媒体内容    [选题生成] 土拍「后海中心区」触发自动生成2条选题
客户会面 → 全板块        [会面brief生成] 为客户「陈总」生成会面brief
```

博主不仅享受自动化，还能**看懂每一步为什么发生** —— 这是 FDE 交付物可信任的关键。
